jQuery对象转变成DOM对象

jQuery对象不能使用DOM中的方法，但如果对jQuery提供的方法不熟悉，可以转变为DOM方法

1）jQuery对象是一个类似数组的对象，可以通过index方法得到相应的DOM对象。代码如下

var $cr=$("#cr");   //jQuery对象

var cr=$cr[0];   //DOM对象

alert(cr.checked);   //检测这个checkbox是否被选中了

2）另一种方法是jQuery本身提供的，通过get(index)方法得到相应的DOM对象。代码如下

var $cr=$("#cr");   //jQuery对象

var cr=$cr.get(0);   //DOM对象

alert(cr.checked);   //检测这个checkbox是否被选中了



DOM对象转变成jQuery对象

对于一个DOM对象，只需要用$()把DOM包装起来，就可以获得一个jQuery对象了。代码如下

var cr=document.getElementById("cr");   //获得DOM对象

var $cr=$(cr);   //获得jQuery对象




强调：DOM对象才可以使用DOM方法，jQuery不能使用DOM方法



例子：




其使用jQuery判定的方法代码为:

$(function(){

  $cr=$("#cr");

  $cr.click(function(){

       if ($cr.is(":checked")) {   //is(":checked")是jQuery中的方法，判断jQuery对象有没有被选中，返回布尔值

         alert("感谢您的支持，您可以继续操作!");

      };

  })

})
